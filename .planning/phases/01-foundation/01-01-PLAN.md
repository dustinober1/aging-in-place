  <name>Task 2: Build encryption and keychain services</name>
  <files>

  </files>
  <action>
    Create the CryptoKit encryption layer following patterns from 01-RESEARCH.md:

    **KeychainService.swift** — `struct KeychainService` with:
    - `static func storeKey(_ key: SymmetricKey, for category: PermissionCategory) throws` — SecItemDelete then SecItemAdd with `kSecAttrAccessibleAfterFirstUnlock`. Service tag: "com.agingInPlace.carekeys". Throws `KeychainError.storeFailed(OSStatus)` on failure.
    - `static func loadKey(for category: PermissionCategory) throws -> SymmetricKey` — SecItemCopyMatching, returns `SymmetricKey(data:)`. Throws `KeychainError.notFound` if absent.
    - `static func loadOrCreateKey(for category: PermissionCategory) throws -> SymmetricKey` — Try loadKey, if `.notFound` generate `SymmetricKey(size: .bits256)`, store, return.
    - `static func deleteKey(for category: PermissionCategory) throws` — SecItemDelete for cleanup/testing.
    - `enum KeychainError: Error` with `storeFailed(OSStatus)`, `notFound`, `deleteFailed(OSStatus)`.

    **EncryptionService.swift** — `struct EncryptionService` with:
    - `static func seal(_ plaintext: Data, for category: PermissionCategory) throws -> Data` — loads key via KeychainService, creates `AES.GCM.seal(plaintext, using: key)`, returns `sealedBox.combined!`.
    - `static func open(_ ciphertext: Data, for category: PermissionCategory) throws -> Data` — loads key, creates `AES.GCM.SealedBox(combined: ciphertext)`, opens with key.
    - `static func rotateKey(for category: PermissionCategory) throws` — generates new `SymmetricKey(size: .bits256)`, stores via KeychainService (overwrites old). Old key is permanently lost.

    Import CryptoKit and Security framework. All methods are synchronous (Keychain calls are fast). Do NOT store keys in UserDefaults or SwiftData — Keychain only.
